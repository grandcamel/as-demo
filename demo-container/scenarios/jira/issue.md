# Issue Management

Create, update, and manage JIRA issues.

## Example Prompts

### Create an Issue
```
Create a bug in project DEMO: "Login button not working on mobile"
```

### Get Issue Details
```
Show me the details of issue DEMO-123
```

### Update an Issue
```
Change the priority of DEMO-123 to High
```

### Transition an Issue
```
Move DEMO-123 to "In Progress"
```

## Tips

- Always specify the project key (e.g., DEMO)
- Use issue keys for precise operations
- You can search for issues first using JQL
