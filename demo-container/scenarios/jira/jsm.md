# JIRA Service Management

Handle service desk requests and workflows.

## Example Prompts

### Create Service Request
```
Create a support ticket: "Need access to production database"
```

### View Requests
```
Show me all open service requests
```

### SLA Status
```
What requests are about to breach SLA?
```

### Approve Request
```
Approve request HELP-456
```

## Tips

- Service requests follow specific workflows
- SLAs are configured per request type
- Approvers can approve or decline requests
