# Support Engineer

Log analysis, troubleshooting, and customer issue investigation.

## Example Prompts

### User Logs

```
Find all logs for user ID 12345
```

### Error Investigation

```
Search for the error "connection refused" in the last hour
```

### Transaction Trace

```
Trace transaction ID abc-123 across all services
```

### Performance Issues

```
Find slow requests taking more than 5 seconds
```

## Tips

- Use user IDs and transaction IDs for tracing
- Check for related errors before and after
- Look at service dependencies
