# DevOps Engineer

Deployment monitoring, CI/CD pipeline insights, and infrastructure health.

## Example Prompts

### Deployment Status

```
Show me the status of recent deployments
```

### Build Failures

```
Find failed CI/CD builds in the last 24 hours
```

### Infrastructure Health

```
What's the health of our Kubernetes clusters?
```

### Log Analysis

```
Search for errors in the deployment logs
```

## Tips

- Use time ranges to narrow searches
- Filter by environment (dev, staging, prod)
- Look for patterns in error logs
