# SRE / On-Call

Alert management, incident response, and system reliability.

## Example Prompts

### View Alerts

```
Show me all critical alerts in the last 4 hours
```

### Error Patterns

```
Find 500 errors in the payment service
```

### System Health

```
What's the error rate across all services?
```

### Incident Investigation

```
Help me investigate the cause of high latency on the API
```

## Tips

- Start with time-based searches
- Correlate errors with deployments
- Check for upstream dependencies
