# Page Hierarchy

Navigate and manage page trees in Confluence.

## Example Prompts

### View Page Tree

```
Show me the page tree for space CDEMO
```

### Get Child Pages

```
List all child pages under "Getting Started"
```

### Move a Page

```
Move page "Old Notes" to be a child of "Archive"
```

## Tips

- Every space has a root page
- Pages can be nested multiple levels deep
- Use page hierarchy for logical organization
