# Page Management

Create, read, update, and delete Confluence pages.

## Example Prompts

### Create a Page

```
Create a new page called "Meeting Notes" in space CDEMO
```

### Read a Page

```
Show me the content of the Getting Started page in CDEMO
```

### Update a Page

```
Update page 12345 to add a new section about project updates
```

### Delete a Page

```
Delete the page titled "Old Notes" from space CDEMO
```

## Tips

- Always specify the space key (e.g., CDEMO)
- Use page IDs for precise operations
- You can search for pages by title first
